UNO Flip – Java FXGL

Implementación digital del juego de cartas UNO Flip desarrollada en Java utilizando el framework FXGL. El proyecto simula la mecánica del juego original, incluyendo cartas especiales, sistema de turnos y el cambio entre el lado claro y oscuro del mazo.

Descripción

UNO Flip es una variante del clásico juego UNO que introduce una mecánica adicional: cada carta tiene dos lados (claro y oscuro). Durante el juego, ciertas cartas permiten cambiar el lado activo del mazo, modificando las reglas y efectos disponibles.

Este proyecto implementa la lógica del juego y su funcionamiento dentro de un entorno interactivo desarrollado con FXGL.

Características

Implementación de las reglas del juego UNO Flip

Sistema de turnos entre jugadores

Cartas especiales con efectos únicos

Mecánica de cambio entre lado claro y lado oscuro

Gestión del mazo y descarte

Interfaz gráfica utilizando JavaFX / FXGL

Tecnologías utilizadas

Lenguaje: Java

Framework: FXGL

Motor gráfico: JavaFX
